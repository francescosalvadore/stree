Understading output files¶

Restart files¶

In order to restart a simulation STREAmS needs at least two files, rst.bin and field_info.dat.

The file rst.bin contains the flow field of the conserved variables. For a single specie ideal gas, these are \(\rho\), \(\rho\,u\), \(\rho\,v\), \(\rho\,w\), and \(\rho\,E\), where \(\rho\) is the density, \(u,v,w\) are the streamwise, wall-normal and spanwise velocity components, and \(E\) is the total energy.

The file field_info.dat contains the following information needed for restart:

  • current step number

  • current time

  • running index for runtime averaging

  • time interval from when last restart file was written

  • time interval from when last statistics file was written

  • time interval from when last slices where written

  • number of flow samples that have been stored in VTK or Plot3D format

The frequency with which rst.bin and field_info.dat are written is controlled by the input parameter dtsave_restart. At each restart, after reading the two files, STREAmS automatically renames them into rst.bak and field_info.bak.

Flow fields¶

Flow fields can be stored in standard VTK and/or Plot3D formats. Plot3D files are constituted by the mesh, stored in plot3dgrid.xyz and the flow fields, stored in files flow_nnnn.q. VTK files are store in flow_nnnn.vtr, which contains both the mesh and the solution. nnnn is the file number. Both Plot3d and VTK files contain the same primitive variables, namely \(\rho\), \(u\), \(v\), \(w\) and \(T\).

Solutions is VTK format are only printed if the input flag enable_vtk=1 and Plot3D files are printed if enable_plot3d=1. The frequency with which these solutions are stored is controlled by the variable dtsave in the input file.

Mean flow statistics¶

Flow statistics are collected at runtime and stored in stat.bin and stat3d.bin. The file stat.bin contains statistics averaged in time and in the spanwise direction, for a total of 68 variables:

  1. \(\overline{\rho}\quad\quad\) 21. \(\overline{\nu}\quad\quad\) 41. \(\overline{p u}\quad\quad\) 61. \(\overline{pu_x}\)

  2. \(\overline{u}\quad\quad\) 22. \(\overline{\omega_x^2}\quad\quad\) 42. \(\overline{p v}\quad\quad\) 62. \(\overline{pv_y}\)

  3. \(\overline{v}\quad\quad\) 23. \(\overline{\omega_y^2}\quad\quad\) 43. \(\overline{\sigma_{11}}\quad\quad\) 63. \(\overline{pw_z}\)

  4. \(\overline{w}\quad\quad\) 24. \(\overline{\omega_z^2}\quad\quad\) 44. \(\overline{\sigma_{12}}\quad\quad\) 64. \(\overline{u_y+v_x}\)

  5. \(\overline{p}\quad\quad\) 25. \(\overline{\rho T}\quad\quad\) 45. \(\overline{\sigma_{13}}\quad\quad\) 65. \(\overline{(\boldsymbol\nabla\cdot\mathbf{U})^2}\)

  6. \(\overline{T}\quad\quad\) 26. \(\overline{\rho T^2}\quad\quad\) 46. \(\overline{\sigma_{22}}\quad\quad\) 66. \(\overline{\rho T^3}\)

  7. \(\overline{\rho^2}\quad\quad\) 27. \(\overline{\rho^2}\quad\quad\) 47. \(\overline{\sigma_{23}}\quad\quad\) 67. \(\overline{\rho T^4}\)

  8. \(\overline{u^2}\quad\quad\) 28. \(\overline{T_0}\quad\quad\) 48. \(\overline{\sigma_{33}}\quad\quad\) 68. \(\overline{\rho u^4}\)

  9. \(\overline{v^2}\quad\quad\) 29. \(\overline{\rho T_0}\quad\quad\) 49. \(\overline{\sigma_{11}u}\)

  10. \(\overline{w^2}\quad\quad\) 30. \(\overline{\rho u T}\quad\quad\) 50. \(\overline{\sigma_{12}u}\)

  11. \(\overline{p^2}\quad\quad\) 31. \(\overline{\rho v T}\quad\quad\) 51. \(\overline{\sigma_{21}v}\)

  12. \(\overline{T^2}\quad\quad\) 32. \(\overline{\rho w T}\quad\quad\) 52. \(\overline{\sigma_{22}v}\)

  13. \(\overline{\rho u}\quad\quad\) 33. \(\overline{M}\quad\quad\) 53. \(\overline{\sigma_{31}w}\)

  14. \(\overline{\rho v}\quad\quad\) 34. \(\overline{M^2}\quad\quad\) 54. \(\overline{\sigma_{32}w}\)

  15. \(\overline{\rho w}\quad\quad\) 35. \(\overline{\rho u^3}\quad\quad\) 55. \(\overline{\sigma_{11}v+\sigma_{21}u}\)

  16. \(\overline{\rho u^2}\quad\quad\) 36. \(\overline{\rho v u^2}\quad\quad\) 56. \(\overline{\sigma_{12}v+\sigma_{22}u}\)

  17. \(\overline{\rho v^2}\quad\quad\) 37. \(\overline{\rho u v^2}\quad\quad\) 57. \(\overline{\sigma_{11}u_x+\sigma_{12}u_y + \sigma_{13}u_z}\)

  18. \(\overline{\rho w^2}\quad\quad\) 38. \(\overline{\rho v^3}\quad\quad\) 58. \(\overline{\sigma_{21}v_x+\sigma_{22}v_y + \sigma_{23}v_z}\)

  19. \(\overline{\rho uv}\quad\quad\) 39. \(\overline{\rho u w^2}\quad\quad\) 59. \(\overline{\sigma_{31}w_x+\sigma_{32}w_y + \sigma_{33}w_z}\)

  20. \(\overline{\mu}\quad\quad\) 40. \(\overline{\rho v w^2}\quad\quad\) 60. \(\overline{\sigma_{31}v_x+\sigma_{12}(u_x+v_y) + \sigma_{22}u_y+\sigma_{13}v_z+\sigma_{23}u_z}\)

The symbol \(\overline{(\cdot)}\) indicates averaging in time and spanwise direction.

  2. \(\langle\rho u\rangle\quad\quad\)

  5. \(\langle p\rangle\quad\quad\)

  6. \(\langle\rho^2\rangle\quad\quad\)

  7. \(\langle\rho u^2\rangle\quad\quad\)

  8. \(\langle\rho v^2\rangle\quad\quad\)

  9. \(\langle\rho w^2\rangle\quad\quad\)

  10. \(\langle\rho u v\rangle\quad\quad\)

  11. \(\langle\rho u w\rangle\quad\quad\)

  12. \(\langle\rho v w\rangle\quad\quad\)

  13. \(\langle\rho T^2\rangle\quad\quad\)

  14. \(\langle p^2\rangle\quad\quad\)

The output frequency of the statistics files is controlled by the input parameter dtsave_restart. If restart_type=0 stat.bin and stat3d.bin are read and renamed stat.bak and stat3d.bak.

progress.out¶

The file progress.out is written and updated at runtime and contains different quantities depending on the initial flow condition.

For channel flow cases it contains the following quantities:

  • step number

  • time step

  • time

  • residual of :math: rho u

  • pressure gradient :math:Pi

  • bulk density

  • bulk velocity

  • bulk temperature

For boundary layer flow cases it contains the following quantities:

  • step number

  • time step

  • time

  • residual of :math: rho u

The same variables printed in progress.out are also printed on screen while the solver is running.
